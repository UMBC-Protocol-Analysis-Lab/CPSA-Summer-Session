# Homework 5

## Instructions
SCEP is complicated, you have **one week** to model it.
Please do not hesitate to ask questions! \
You will be modeling a component of a real protocol called SecureDNA, developed to facilitate secure transfer of DNA information between clients.
In the directory [./homework5files](homework5files) you will find a simplified specification for SCEP.
Create a CPSA model of SCEP using the simplified [spec](homework5files/spec.md).
Run CPSA and see if you can spot any errors in the protocol's design.

## Deliverable
1. Your model of SCEP in CPSA.
    1. Submit your `*.scm` file
2. Write a full page describing the following
   1. What the system is intending to do
   2. What security flaws are present in the system
   3. Any fixes that you can produce (It is OK if you cannot figure out how to fix all the errors)
