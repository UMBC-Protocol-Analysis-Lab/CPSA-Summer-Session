# Homework 5

## Instructions
You will be modeling a real protocol called SecureDNA, developed to facilitate secure transfer of DNA information between clients.
In the directory [./homework5files](homework5files) you will find a simplified specification for SecureDNA and some starter files to help you.
Using the simplified spec and the starter files, create a CPSA model of SecureDNA.
If you would like an additional challenge, do not use the starter files, simply start with an empty file in a fresh directory.
Run CPSA and see if you can spot any errors in the protocol's design.

## Deliverable
1. Your model of SecureDNA.
2. A list of any errors made in the design of SecureDNA.
