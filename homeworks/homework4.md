# Homework 4

## Instructions
You will be modeling a real protocol that is currently in use called FIDO UAF.
In the directory [./homework4files](homework4files) you will find a simplified specification for FIDO, and some starter files to help you get started.
Using the simplified spec and the starter files, create a CPSA model of FIDO UAF.
Run CPSA and see if you can spot any errors in the protocol's design.

## Deliverable
1. Your model of FIDO UAF.
2. A list of any errors made in the design of FIDO.
